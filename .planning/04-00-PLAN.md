# Plan 04-00: HUD Visual System

## Goal
Unify the entire UI under the "Music of the Spheres" visual language: high-contrast yellow panels, geometric sans-serif typography, and a consistent symbol/divider system across MainFrame, SideStrip, and all dashboard overlays.

## Requirements
- **UX-05** (new): Visual coherence across all HUD surfaces
- Implicitly supports UX-01 through UX-04 by establishing design tokens before functional work

## Success Criteria
1. [x] All dashboard overlays (planet + belt) render with yellow content cards on dark panels
2. [x] MainFrame header/footer typography uses geometric sans-serif (Space Grotesk) with tracking
3. [x] SideStrip adopts asymmetric border radius and unified dividers
4. [x] Google Font (Space Grotesk + Inter + JetBrains Mono) loads without FOIT/FOUT issues
5. [x] No regressions to planet orbit animation or click interactions

## Design Tokens Added
### Typography
- `--font-header`: Space Grotesk (800 weight headers)
- `--font-body`: Inter (400/700)
- `--font-mono`: JetBrains Mono (labels, CLI)

### Colors
- `--hud-void`: #030305
- `--hud-panel-dark`: rgba(10, 10, 20, 0.95)
- `--hud-panel-yellow`: #FCE96A
- `--hud-text-primary`, `--hud-text-inverse`, `--hud-text-muted`
- `--hud-border-subtle`, `--hud-border-yellow`
- `--hud-accent-yellow`: #FCE96A

### Spacing & Radii
- `--hud-radius-lg`: 0 48px 0 48px
- `--hud-radius-md`: 16px
- `--hud-radius-sm`: 8px

### Shadows & Glows
- `--hud-shadow-card`: 0 20px 50px rgba(0,0,0,0.5)
- `--hud-shadow-glow`: 0 0 15px var(--hud-accent-yellow)
- `--hud-shadow-planet`: 0 0 50px rgba(255,223,0,0.6)
- `--hud-glow-text`: 0 0 10px var(--hud-accent-yellow)

## Risk Mitigation
- **Font loading**: `font-display: swap` via Google Fonts CDN
- **Contrast**: Yellow #FCE96A on black #0a0a15 passes WCAG AA for large text
- **Regression**: All changes are CSS-only; no JS game-loop modifications
- **Bundle size**: Google Fonts via CDN; no local font files added
