7. Render structure (per UI-SPEC BeltHUD Layout Contract):
   ```tsx
   <div className="belt-dashboard-overlay" style={{ "--theme-color": beltDataModel.color }}>
     <div className="belt-dashboard-container">
       {/* Back button — absolute positioned */}
       <div style={{ position: "absolute", zIndex: 20 }}>
         <HudButton variant="back" label="SYSTEM VIEW" themeColor={beltDataModel.color} onClick={onClose} icon={<svg...arrow-left.../>} />
       </div>
       {/* Right column only — full width (no left 3D planet column) */}
       <div className="belt-dashboard-right" style={{ fontFamily: "monospace" }}>
         <CliHeader planetId="ai-belt" color={beltDataModel.color} onComplete={() => setIsFetched(true)} />
         {isFetched && (
           <div className="fetched-data-wrapper">
             <h1 className="planet-title" style={{ color: beltDataModel.color, textShadow: `0 0 15px ${beltDataModel.color}60` }}>
               {beltDataModel.name}
             </h1>
             <div className="planet-debrief" style={{ borderLeft: `2px solid ${beltDataModel.color}` }}>
               <strong style={{ color: "#fff" }}>STATUS:</strong> {beltDataModel.type} <br />
               {beltDataModel.description}
             </div>
             <Tabs tabs={beltTabs} defaultTabId="missions" themeColor={beltDataModel.color} />
           </div>
         )}
       </div>
     </div>
   </div>
   ```
8. Use React.FC pattern with explicit interface, useCallback for handleFetchComplete (per conventions)
