    Fix layout thrashing in the PlanetarySystem animation loop by caching container dimensions instead of reading them every frame.

    Current problem (lines 43-44 in the animate function):
    ```typescript
    const w = containerRef.current.clientWidth;
    const h = containerRef.current.clientHeight;
    ```
    These DOM reads happen every frame (~60 times/second), forcing the browser to recalculate layout each time.

    Fix: Cache dimensions in refs and only update them on resize events.

    Add two new refs at the top of the component (after existing refs):
    ```typescript
    const containerWidthRef = useRef<number>(0);
    const containerHeightRef = useRef<number>(0);
    ```

    Add a ResizeObserver to update cached dimensions when the container resizes:
    ```typescript
    useEffect(() => {
      const container = containerRef.current;
      if (!container) return;

      // Initial measurement
      containerWidthRef.current = container.clientWidth;
      containerHeightRef.current = container.clientHeight;

      const observer = new ResizeObserver((entries) => {
        for (const entry of entries) {
          containerWidthRef.current = entry.contentRect.width;
          containerHeightRef.current = entry.contentRect.height;
        }
      });
      observer.observe(container);

      return () => observer.disconnect();
    }, []);
    ```

    In the animate function, replace the DOM reads with ref reads:
    ```typescript
    // REPLACE these two lines:
    // const w = containerRef.current.clientWidth;
    // const h = containerRef.current.clientHeight;

    // WITH these:
    const w = containerWidthRef.current;
    const h = containerHeightRef.current;
    ```

    Do NOT change any other logic in the animate function. The physics calculations, pixel coordinate mapping, SVG path updates, and DOM transforms must remain exactly as they are.

    Also remove `physicalPlanets` from the useEffect dependency array since the ResizeObserver handles dimension updates independently. Keep `hoveredPlanetId` as the only dependency since it affects the physics (paused animation on hover).
